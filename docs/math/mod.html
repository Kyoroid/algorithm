合同¶

整数を法とする合同¶

Wikipedia

定義¶

整数 \(a, b, m\) に関して

\[a-b \in \{km \mid k=0, \pm 1, \pm 2, \pm 3, \ldots\}\]

が成り立つとき、「整数 \(a, b\) が法 \(m\) に関して合同である」という。

表記¶

「整数 \(a, b\) が法 \(m\) に関して合同である」ことを、次のように表記する。

\[\begin{split}\begin{eqnarray} a &\equiv& b \pmod{m} \\ a &\equiv& b \end{eqnarray}\end{split}\]

などと書く。

加算・乗算・減算・べき乗¶

\(n\)を整数とする。\(a_1 \equiv b_1 , a_2 \equiv b_2\)のとき、加法・乗法が定義される。

\[\begin{split}\begin{eqnarray} a_1 + a_2 &\equiv& b_1 + b_2 \\ a_1 a_2 &\equiv& b_1 b_2 \\ \end{eqnarray}\end{split}\]

加減算、乗算、べき乗は次のようになる。

\[\begin{split}\begin{eqnarray} a \pm x &\equiv& b \pm x && \\ ax &\equiv& bx && \\ a^x &\equiv& b^x && (x>0) \\ \end{eqnarray}\end{split}\]

逆元¶

合同式では \(\gcd(a, m) = 1\) (整数\(a\)と法\(m\)が互いに素) のときに限り、逆元 (モジュラ逆数) \(a^{-1}\)が次のように定義される。

\[a a^{-1} \equiv 1\]

逆元計算¶

逆元の値は拡張ユークリッドの互除法により求めることができる。
逆元の定義より
\[a a^{-1} - 1 \equiv 0 \pmod{m}\]

であるから、これを合同式ではなく等式として表すと

\[\begin{array} a a^{-1} - 1 = mk & (k \in \mathbb{N}) \end{array}\]

となる。これを変形し

\[a a^{-1} + m(-k) = 1\]

とする。ここで制約

\[\gcd(a, m) = 1\]
をおくと、等式は拡張ユークリッドの互除法の式 \(ax + my = \gcd(a, m)\) と一致する。
つまり、 \(\gcd(a, m) = 1\) となる\(a, m\) に対して \(a a^{-1} + m(-k) = 1\) の解が存在し、その解は
\[(x, y) = (a^{-1}, -k)\]

となる。同時に逆元 \(a^{-1}\) が \(x\) の値として求まる。

逆元計算の実装¶

[1]:

import sys
sys.setrecursionlimit(200000)


def extgcd(a, b):
    if a < b:
        m, y, x = extgcd(b, a)
        return (m, x, y)
    if b == 0:
        x = 1
        y = 0
        return (a, x, y)
    m, y, x = extgcd(b, a % b)
    y = y - a // b * x
    return (m, x, y)


def inv(a, m):
    n, x, y = extgcd(a, m)
    if n != 1:
        raise ValueError("{} and {} are not mutually prime.".format(a, m))
    return x

\(\gcd(a, m) = 1\) が成り立つときは、逆元 \(a^{-1}\) を返す。

[2]:

import math

a = 10
m = 2019
print("gcd({a}, {m}) = {d}".format(a=a, m=m, d=math.gcd(a, m)))

a_inv = inv(a, m)
print("{a}^-1 \u2261 {d} (mod {m})".format(a=a, m=m, d=a_inv))
print("{a} * {a}^-1 \u2261 {d} (mod {m})".format(a=a, m=m, d=a * a_inv % m))

gcd(10, 2019) = 1
10^-1 ≡ 202 (mod 2019)
10 * 10^-1 ≡ 1 (mod 2019)

\(\gcd(a, m) = 1\) が成り立たない場合は例外を送出する。

[3]:

import math

a = 10
m = 2020
print("gcd({a}, {m}) \u2261 {d}".format(a=a, m=m, d=math.gcd(a, m)))

try:
    inv(a, m)
except ValueError as e:
    print(e)

gcd(10, 2020) ≡ 10
10 and 2020 are not mutually prime.

除算¶

除算 \(a \div b\) は、逆元との積として定義される。
逆元の定義より、 \(\gcd(b, m) = 1\) となる整数 \(b\) に対してのみ除算が可能である。
\[a \div b \equiv a \times b^{-1}\]

素数を法とする合同¶

逆元¶

法 \(p\) が素数で \(\gcd(a, p)=1\) を満たす場合は、拡張ユークリッドの互除法を使わずに、累乗により逆元を求めることもできる。

整数 \(a\) と互いに素である素数 \(p\) をおく。フェルマーの小定理

\[a^{p-1} \equiv 1 \pmod{p}\]

より、逆元に関する次の合同式が成り立つ。

\[\begin{split}\begin{eqnarray} a a^{-1} &\equiv& a a^{p-2} \equiv 1 \pmod{p} \\ a^{-1} &\equiv& a^{p-2} \pmod{p} \\ \end{eqnarray}\end{split}\]

逆元計算の実装¶

Pythonにおいては、 powの第三引数を用いて累乗のmodを計算できる。

[4]:

def pinv(a, p):
    return pow(a, p-2, p)

[5]:

a = 131
m = 10**9+7
a_pinv = pinv(a, m)
print("{a}^-1 \u2261 {d} (mod {m})".format(a=a, m=m, d=a_pinv))

ModInt¶

operator overload

法 \(m\) における整数をModIntとして定義する。
intはイミュータブルオブジェクトなので、__init__では初期化できないことに注意。
[7]:

class ModInt(int):

    MOD = 10**9+7

    def __new__(cls, x, *args, **kwargs):
        return super().__new__(cls, x % ModInt.MOD)

    def __add__(self, other):
        return ModInt(super().__add__(other) % ModInt.MOD)

    def __radd__(self, other):
        return ModInt(super().__radd__(other) % ModInt.MOD)

    def __mul__(self, other):
        return ModInt(super().__mul__(other) % ModInt.MOD)

    def __rmul__(self, other):
        return ModInt(super().__rmul__(other) % ModInt.MOD)

    def __pow__(self, other, *args):
        return ModInt(super().__pow__(other, ModInt.MOD))

    def __rpow__(self, other, *args):
        raise NotImplementedError()

    def __truediv__(self, other):
        inv = ModInt(other).inverse()
        return ModInt(super().__mul__(inv) % ModInt.MOD)

    def __floordiv__(self, other):
        return self.__truediv__(other)

    def __rtruediv__(self, other):
        return ModInt(self.inverse().__mul__(other))

    def __rfloordiv__(self, other):
        return self.__rtruediv__(other)

    def _extgcd(self, a, b):
        assert not isinstance(a, ModInt) and not isinstance(b, ModInt)
        if a < b:
            m, y, x = self._extgcd(b, a)
            return (m, x, y)
        if b == 0:
            x = 1
            y = 0
            return (a, x, y)
        m, y, x = self._extgcd(b, a % b)
        y = y - a // b * x
        return (m, x, y)

    def inverse(self):
        a = int(self)
        m = int(ModInt.MOD)
        n, x, y = self._extgcd(a, m)
        if n != 1:
            raise ValueError("{} and {} are not mutually prime.".format(a, m))
        return ModInt(x)
